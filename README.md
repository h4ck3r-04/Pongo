# Pongo

Mongodb Driver for Perl

## Installation

For Linux

```bash
sudo apt-get install libmongoc-1.0-0 libbson-1.0-0 libmongoc-dev libbson-dev
```

For Macos

```zsh
brew install mongo-c-driver
```

## Development Setup

To build the package

```bash
perl Makefile.PL
make
make install
```

To Clean the Build

```bash
make clean
```
